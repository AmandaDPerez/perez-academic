Project: Impact of ART on Weathering

Data Information

Outcomes: Allostatic Load, named AL_75_mix and is a function of:

  • IL6 (75th percentile),
  • epinephrine (75th percentile),
  • norepinephrine (75th perc.),
  • cortisol (75th perc.),
  • hsCRP (JNC7),
  • diastolic blood pressure (JNC7),
  • systolic blood pressure (JNC7),
  • HDL (ATPIII),
  • LDL (ATPIII),
  • waist (ATPIII),
  • glucose (Expert Committee on Diagnosis and Classification of Diabetes Mellitus),
  • HBA1C (Expert Committee on Diagnosis and Classification of Diabetes Mellitus),
  • cholesterol (ATPIII)
  • triglycerides (ATPIII)
  • BMI (standards)

Binary Allostatic Load, named AL_75_dichot is a binary version that sets (AL_75_mix < 6) = 0 and (AL_75_mix >= 6) = 1

Telomere length (n = 140) named tlr. Has been completely merged with data

Predictor: Anticipatory Racism Threat aRT

Reliability of the subscales:

  • art_resilience: alpha = 0.66 (0.59, 0.74)
  • art_confirmation_concern: alpha = 0.94 (0.93, 0.95)
  • art_stereotype_awareness: alpha = 0.86 (0.83, 0.89)

# For analysis use drinksperday (not alcohfreq_yr) as the covariate. 
# Standards show that for women of all ages up to 1 drink per day can have health benefits. 
# This is considered moderate use. Anything more than that is considered a health risk. 
# RE: DRINKS PER DAY. THE QUESTION ASKS ON A TYPICAL DAY WHEN YOU DRANK, HOW MANY DRINKS DID YOU HAVE. 
# THEREFORE, IT SHOULD BE CODED 0 FOR <=3 DRINKS AND 1 FOR >3 DRINKS. THIS IS CONSISTENT WITH STANDARDS 
# INDICATING THAT >3 DRINKS ON ANY GIVEN DAY IS CONSIDERED HEAVY DRINKING. 
# BUT FORDRINKSPERDAY 1=1, 2=1-2, 3=3-4, 4=5-6 AND SO ON. SO CAN'T CUT IT EXACTLY AT 3. 
#  THEREFORE, CODE AS 0 FOR 1 AND 2 AND CODE 1 FOR >=3.
dta_long <- dta_long %>% 
  mutate(alcohuse = ifelse(drinksperday <= 2, 0, 1))

# This var is messed up because the high numbers (highest is 45) don't make sense based on the question 
# (categories 0-7). Since the standard is 30 min at least 5 days per week, could cut it at 0=>=5 & 1=<5.
# For exercise, 0=healthy and 1=not healthy based on current standards of exercising a minimum of 5 times 
# per week for 30 minutes.
dta_long <- dta_long %>% 
  mutate(exercise = ifelse(exercise_week >= 5 & exercise_week <= 45, 0, 1))

         p_systolbp = exp(ln_systolbp),
         p_diastolbp = exp(ln_diastolbp)) %>% 
  mutate(
    # hsCRP
    # *AHA guidelines for hsCRP
    # <1=low risk, 1-3=average risk, >3=high risk.
    hscrp_cat = case_when(p_hscrp < 1 ~ 0,
                          p_hscrp >= 1 & p_hscrp <= 3 ~ 1,
                          p_hscrp > 3 & p_hscrp <= 40 ~ 2),
    hscrp_dichot = case_when(p_hscrp <= 3 ~ 0,
                             p_hscrp > 3 & p_hscrp <= 40 ~ 1),
    # blood pressure
    diastolbp_cat_jnc7 = case_when(p_diastolbp < 80 ~ 0,
                                   p_diastolbp >= 80 & p_diastolbp < 90 ~ 1,
                                   p_diastolbp >= 90 & p_diastolbp <= 151 ~ 2),
    systolbp_cat_jnc7 = case_when(p_systolbp < 120 ~ 0,
                                  p_systolbp >= 120 & p_systolbp < 140 ~ 1,
                                  p_systolbp >= 140 & p_systolbp <= 194 ~ 2),
    #JNC8 says <90 for clinical target for not at risk. We are following the JNC7 guidelines that indicate pre-hypertension -- so disease process already started. Challenge is, if these people not really at risk, could bias results towards the null. So will create a second variable using JNC8.
    diastolbp_dichot_jnc7 = ifelse(p_diastolbp < 80, 0, 1),
    diastolbp_dichot_jnc8 = ifelse(p_diastolbp < 90, 0, 1),
    systolbp_dichot_jnc7 = ifelse(p_diastolbp < 120, 0, 1),
    systolbp_dichot_jnc8 = ifelse(p_diastolbp < 140, 0, 1),
    # HDL guidelines from ATPIII
    hdl_dichot = ifelse(p_hdl >= 50 & p_hdl <= 152, 0, 1),
    hdl_dichot_90 = ifelse(p_hdl >= 40 & p_hdl <= 152, 0, 1),
    # ATPIII measures high-risk for women is <50 and men <40. Low risk is measured at >=60 in general. 
    # hdl is coded such that higher levels of hdl are better for health and lower numbers are worse for 
    # health. Therefore 0-2 goes in order of increaseing risk with 2 representing the lowest hdl numbers, 
    # and correspondingly the highest level of risk.
    hdl_cat = case_when(p_hdl >= 50 & p_hdl < 153 ~ 0,
                        p_hdl >= 40 & p_hdl < 50 ~ 1,
                        p_hdl < 40 ~ 2),
    # TG guidelines use ATPIII
    triglyc_dichot = ifelse(p_triglyc < 150, 0, 1),
    triglyc_dichot_90 = ifelse(p_triglyc < 200, 0, 1),
    triglyc_cat = case_when(p_triglyc < 150 ~ 0,
                            p_triglyc >= 150 & p_triglyc < 200 ~ 1,
                            p_triglyc >= 200 & p_triglyc <= 499 ~ 2), # *official risk categories for triglycerides has one additional level of risk (>=500), but we had no values this high.
    # ATPIII guidelines
    waist_dichot = ifelse(waist <= 35, 0, 1), #ATPIII guidelines
    waisthip = waist/hip,
    waisthip_dichot = ifelse(waisthip <= 0.85, 0, 1), #*WHO guidelines ATPIII doesn't have a guidelne for waist/hip.
    # AIC guidelines based on Expert Committee on Diagnosis and Classification of Diabetes Mellitus (standards of med care in diabetes, 2011)
    hba1c_dichot = ifelse(p_hba1c < 5.7, 0, 1),
    hba1c_dichot_90 = ifelse(p_hba1c < 6.5, 0, 1),
    hba1c_cat = case_when(p_hba1c < 5.7 ~ 0,
                          p_hba1c >= 5.7 & p_hba1c < 6.5 ~ 1,
                          p_hba1c >= 6.5 & p_hba1c < 11.5 ~ 2),

    hscrp_90 = ifelse(p_hscrp <= 9.64, 0, 1),
    # Epinephrine
    epi_75 = ifelse(p_epinephrine <= 77.70, 0, 1),
    epi_90 = ifelse(p_epinephrine <= 120, 0, 1),
    # Norepi
    norepi_75 = ifelse(p_norepi <= 686.30, 0, 1),
    norepi_90 = ifelse(p_norepi <= 848.95, 0, 1),
    # Cortisol
    cort_75 = ifelse(p_cortisol <= 12.69, 0, 1),
